# Level Zero - RPG Game

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Java](https://img.shields.io/badge/Java-22%2B-orange)

---

**Level Zero** is a Java-based RPG game focused on combat, and gearing up.

---

## Features

- Combat system
- Character selection
- Inventory system

---

## Getting Started

### Requirements
- Java 22 or higher
- Git

### Run the game

```bash
git clone https://github.com/Zeteox/Level-Zero.git
cd src/com/levelzero
javac Main.java
java Main
```
